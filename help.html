Space Trace Plugin for QGIS

Installation of Dependencies

  1. Open the OSGeo4W Shell (included with QGIS installation).
  2. Run the following commands to install the required libraries: 
    python3 -m pip install pyorbital -U --user
python3 -m pip install spacetrack -U --user
python3 -m pip install poliastro -U --user

Features

  • Data Sources: Fetch orbital data (TLE or OMM) from the SpaceTrack API or load from a local file.
  • Track Generation: Create point and line layers representing a satellite's path for a selected date.
  • Customization: Configure the time step (in minutes) for orbital path calculation.
  • Output Options:
    • Save output as Shapefile, GeoPackage, or GeoJSON.
    • Add generated layers directly into the QGIS project.
  • Data Saving: Optionally save TLE/OMM data retrieved from the API.
  • Logging: Monitor processing steps and results via the log tab.

Usage

  1. Open the plugin from the Vector menu: Space Trace > Draw flight path.
  2. Select the data source: local file or SpaceTrack API.
  3. Configure date and time step for the orbit track.
  4. Specify output format and file path (optional).
  5. Enable saving options as needed.
  6. Click Execute to generate the track.

Satellite Search and Query Builder

  • Search Modes:
    • By Name
    • All Active Satellites
    • By Country Code (e.g., US)
    • By NORAD ID (single, range, or list)
  • Custom Query Builder:
    • Select any field (e.g., Launch Date, Eccentricity)
    • Choose comparison operators (=, !=, <, >, LIKE)
    • Enter values with automatic input hints
    • Combine multiple conditions
  • Result Table: Sortable columns with satellite info. When you select a line and click OK, the norad ID of the selected satellite is copied to the main dialog.
